# AI First CRM

AI-powered CRM for Healthcare Professionals.

## Tech Stack

- React (Vite)
- FastAPI
- PostgreSQL
- SQLAlchemy
- Pydantic

## Features

- View Healthcare Professionals
- AI-powered interaction summaries
- Interaction history
- PostgreSQL database
- REST APIs

## Backend

```bash
cd backend
uvicorn main:app --reload
```

## Frontend

```bash
cd frontend
npm install
npm run dev
```
